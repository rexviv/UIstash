# UIstash 运行说明

## 项目简介
UIstash 是一个 Chrome 插件，用来把优秀网页保存为本地版本化快照，并提供全文搜索、标签、备注、筛选和 ZIP 导出能力。

## 启动开发构建
1. 在项目根目录打开 PowerShell。
2. 安装依赖：`npm install`
3. 启动监听构建：`npm run dev`
4. 构建产物会输出到 `dist` 目录。

## 在 Chrome 中加载插件
1. 打开 Chrome，进入 `chrome://extensions/`
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择项目里的 `dist` 目录。
5. 安装后点击工具栏里的 `UIstash` 图标即可使用。

## 重新构建
- 日常开发时保持 `npm run dev` 运行即可，它会在文件变化后重新输出到 `dist`。
- 如果只想做一次生产构建，使用：`npm run build`

## 运行测试
- 执行：`npm run test`

## 如何停止项目
- 如果当前在运行 `npm run dev`，在终端按 `Ctrl + C` 停止监听构建。
- 如果已经加载了扩展，回到 `chrome://extensions/`，对 `UIstash` 点击“移除”或关闭启用开关即可停用。

## 使用建议
- 第一次使用请先在插件弹窗或管理页中选择本地归档目录。
- 如果浏览器重启后目录授权失效，重新授权目录后点击“重试待补写队列”即可把暂存任务补写到本地。
